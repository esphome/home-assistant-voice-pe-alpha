# Home Assistant Voice: Preview Edition

**This is an alpha version of the firmware for testing experimental new features. See [this repository](https://github.com/esphome/home-assistant-voice-pe/) for the current stable version!**

See [the documentation](https://voice-pe.home-assistant.io/) for set up and troubleshooting.

If you need to re-install the firmware, [use this installer](https://esphome.github.io/home-assistant-voice-pe-alpha/).
